#### List当中冒号：的作用

用于定义分片、步长。

`a[ : n]`表示从第0个元素到第n个元素(**不包括n**)，`a[1: ] `表示该列表中的第1个元素到最后一个元素。主要牢记下面的这个语法即可：

```python
arr[开始位置:结束位置:迭代步数]
```

其中每个值都可以省略，具体的规则是：

- **省略开始位置值：则默认值是遍历顺序上的第一个位置。**比如说`arr[:3]`就是从第0个元素到第3个元素(不包括第三个元素,这里可以理解成Python常见的**左闭右开**原则.)

- **省略结束位置值：则默认值是遍历顺序上最后一个位置。**比如说`arr[1:]`就是List中第1个元素到最后一个元素;

- **省略迭代步数（通常是不写第二个冒号），默认值是1（顺序的步数为1的遍历）**,比如说`arr[1:3]`就是List中第1,2这两个元素;

  

其中需要注意以下几点:

- 冒号前后指定位置的变量或者常量还有更灵活的用法，例如可以指定为负数，例如 `arr[2:-2]`,这里的-2指的是倒数第2个位置;本质上是用 `len(arr)+1-2` 计算得位置。
- 如果步数被指定为了负数,则代表从后往前进行遍历.比如说下面的这个例子:

```python
arr = [1, 2, 3, 4, 5, 6, 7]
print(arr[-2::-1])
```
